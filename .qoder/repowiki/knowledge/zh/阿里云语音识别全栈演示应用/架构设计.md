根目录作为单一 Node.js 入口（server.js），将两层职责合并在同一进程中：
- 静态资源层：`express.static(__dirname)` 直接暴露 `index.html`、`js/`、`css/`、`fonts/`、`img/`，浏览器端由 `app.js` + `speech.js` + `aliyun-speech.js` 驱动录音与 WebSocket 流式识别。
- API 层：在 `/api/token` 路由中用 `@alicloud/pop-core` 调用阿里云 NLS CreateToken，AccessKey 仅从 `.env` 读取，不暴露给前端；错误码统一包装为 `{ success, message }` JSON 响应供前端消费。
- 配置通过 dotenv 注入，端口默认 3000，`npm start` 即同时提供静态站点与 token 服务，无需额外反向代理。
