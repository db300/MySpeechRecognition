- **入口与编排**：`index.html` 作为单页入口，通过 `<script type="module">` 加载 `js/app.js` 启动应用。
- **模块化设计**：采用 ES Modules 架构，`App` 类负责 UI 状态管理与事件分发，`SpeechRecognition` 类封装底层 Web Speech API（含自动重连与错误处理），`ParticleSystem` 类独立管理 Canvas 粒子动画。
- **样式体系**：`css/style.css` 定义全局 CSS 变量（如霓虹色系）与响应式布局，并通过 `@font-face` 引入自定义字体 `HYNiaoWen260605.ttf`。
- **依赖方向**：UI 层 (`app.js`) 依赖于功能层 (`speech.js`, `particles.js`)，功能层无外部依赖，保持纯净。
