- 单文件脚本架构：`api/token.php` 作为唯一入口，直接处理 HTTP 请求。
- 配置加载：从项目根目录的 `.env` 文件手动解析 `ALIYUN_ACCESS_KEY_ID` 和 `ALIYUN_ACCESS_KEY_SECRET`。
- 签名逻辑：内置 `percentEncode` 和 HMAC-SHA1 签名算法，构造阿里云 POP API 所需的规范化请求字符串。
- 通信方式：使用 PHP cURL 扩展向 `https://nls-meta.cn-shanghai.aliyuncs.com/` 发送 GET 请求。
- 响应处理：解析 JSON 响应，统一返回包含 `success`、`token`、`expireTime` 等字段的标准 JSON 格式，并处理特定错误码（如权限不足）。
