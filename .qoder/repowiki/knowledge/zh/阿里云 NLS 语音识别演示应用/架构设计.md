根目录即整个应用：server.js 通过 express.static(__dirname) 将 index.html、js/、css/、fonts/、img/ 作为静态资源暴露给浏览器；同时在同一 Express 实例上挂载 /api/token 路由，由 api/token.php（实际应为 .js）调用 @alicloud/pop-core 获取阿里云 NLS Token。前后端通过 JSON 契约 { success, message } 通信，AccessKey 仅从 .env 注入后端，不进入前端 bundle。npm start 启动后无需反向代理即可直接访问 http://localhost:3000。
